# 冥想播放器

一个简洁的Android MP3播放器应用，支持目录选择和连续播放功能。

## 功能特性

1. **目录浏览**: 自动扫描并显示"播放文件"目录下的所有包含MP3文件的子目录
2. **连续播放**: 选择目录后，可以连续播放该目录下的所有MP3文件
3. **定时停止**: 支持设置播放时间，到达指定时间后自动停止播放
4. **简洁界面**: 保持界面简洁，易于使用

## 使用说明

1. 将MP3文件放在设备的"播放文件"目录下，按子目录组织
2. 打开应用，选择要播放的目录
3. 点击播放按钮开始播放
4. 可选：设置定时停止时间（分钟）

## 文件结构

应用会在以下位置查找"播放文件"目录：
- `/sdcard/播放文件` (最常见)
- `/storage/emulated/0/播放文件`
- 应用外部存储目录
- 应用内部存储目录

### 如何将文件复制到Android设备

1. **使用USB连接**：
   - 连接Android设备到电脑
   - 在设备上选择"文件传输"模式
   - 将项目中的"播放文件"文件夹复制到设备的根目录或sdcard目录

2. **使用ADB命令**：
   ```bash
   adb push "播放文件" /sdcard/
   ```

3. **直接在设备上操作**：
   - 使用文件管理器在设备上创建"播放文件"目录
   - 将MP3文件按子目录组织后复制到该目录

## 权限要求

- Android 13及以上：需要READ_MEDIA_AUDIO权限
- Android 12及以下：需要READ_EXTERNAL_STORAGE权限

## 构建说明

使用Android Studio打开项目，或使用命令行：

```bash
./gradlew assembleDebug
```

生成的APK文件位于：`app/build/outputs/apk/debug/app-debug.apk`
